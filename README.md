# Portfolio Armin Imamovic

A minimalistic, dark-themed portfolio website showcasing my professional experience, skills, and projects as a Junior Java Developer.

## Features

- **Minimalistic Design**: Clean, modern interface with dark theme
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Subtle animations and transitions for enhanced user experience
- **Interactive Elements**: Hover effects, typing animation, and scroll-triggered animations
- **One-Pager Layout**: All content organized in a single, scrollable page

## Sections

- **Hero**: Introduction with contact information
- **About**: Brief personal overview
- **Experience**: Professional work history timeline
- **Education**: Academic background and training
- **Skills**: Technical skills and competencies
- **Projects**: Featured projects including Sportradar Calendar and JUQE Project
- **Languages**: Language proficiency
- **Contact**: Get in touch information

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS Grid and Flexbox
- **JavaScript**: Interactive features and animations
- **Responsive Design**: Mobile-first approach

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/arminisa95/portfolio_armin_imamovic.git
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio_armin_imamovic
   ```

3. Open `index.html` in your web browser or serve it with a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   ```

4. Visit `http://localhost:8000` to view the portfolio.

## Customization

- Edit `index.html` to modify content
- Update `style.css` to change styling and colors
- Modify `script.js` to adjust animations and interactions
